# Griha Lakshmi Marriage Hall

Official website for **Griha Lakshmi Marriage Hall**, a premium venue for weddings, receptions, and auspicious ceremonies.

## Project Vision

A digital invitation and gallery experience that captures the grandeur of the venue through immersive parallax sections, cinematic video tours, and elegant design.

### Features
- **Cinematic Parallax Hero**: Layered animations featuring the palace in the clouds.
- **Video & Photo Gallery**: Immersive tour of the palace and celebration highlights.
- **Celebration Wheel**: Interactive showcase of events hosted at the venue.
- **Dynamic Contact Widget**: Persistent floating icons for WhatsApp, Call, and Email.
- **Responsive Design**: Tailored for both desktop and mobile royal experiences.

## Tech Stack
- **Frontend**: React + Vite + TypeScript
- **Styling**: Tailwind CSS + Shadcn UI
- **Animations**: GSAP + Framer Motion
- **Icons**: Lucide React

## Development

```sh
# Install dependencies
npm i

# Run development server
npm run dev
```

## Production

```sh
# Build for production
npm run build
```
